Infinity Circles

ITC is a Web3 lifestyle app with SocialFi and GameFi elements
Players can make handsome earnings by cycling for picnics, cycling to exercise, bike rental


Users equip themselves with NFTs in the form of Bikes. By running outdoors, users will earn game currency, which can either be used in-game, or cashed out for profit.

With Game-Fi, ITC aims to nudge millions toward a healthier lifestyle, combat climate change and connect the public to Web 3.0, all while simultaneously hinging on it’s Social-Fi aspect to build a long-lasting platform fostering user generated Web 3.0 content.
